# Constellation_crawler
Scrapy 抓取星座網站資訊
`python run.py`

## Environment
Python 3.6.10
Scrapy 1.6.0
MongoDB v3.6.3

## Schema
|Fields   |Type   |
|---|---|
|_id   |Object   |
|constellations   |Array   |
|date   |Date   |
|love   |String   |
|money   |String   |
|total   |String   |
|work  |String   |
